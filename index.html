Glossary

These are short definitions for some of the most common terms used across the course.

Agent

A system that can pursue a goal by planning, taking actions, checking results, and continuing as needed.

LLM

A large language model trained to predict text patterns across a huge amount of example data.

Prompt

The instructions and context you give the model for the current task.

Context Window

The amount of information the model can consider at once.

Tool Calling

Letting the model choose and use functions such as search, browser, or code execution.

RAG

A pattern where the system fetches relevant information before generating the answer.

Embedding

A numerical representation of meaning used for search and similarity.

Workflow

A repeatable sequence of steps that moves work from start to finish.

Guardrail

A rule or validation that reduces bad outputs or unsafe behavior.

Human-in-the-loop

A design where the system pauses for review at important moments.
